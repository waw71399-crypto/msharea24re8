# معرض مشاريع الطالبات

منصة لعرض مشاريع الطالبات الإبداعية.

## التقنيات

- React + TypeScript + Vite
- Express.js + Tailwind CSS
- دعم كامل للغة العربية (RTL)

## التشغيل

```bash
npm install
npm run dev
```

## النشر على GitHub Pages

راجع [`GITHUB_PAGES_SETUP.md`](./GITHUB_PAGES_SETUP.md) للخطوات الكاملة.

باختصار:
1. ارفع للـ GitHub
2. Settings → Pages → Source: `GitHub Actions`
3. انتهى!

الموقع: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
